
### Colors:
- **Primary Color: Navy Blue (#003366)**
  - Use for headers, primary buttons, and important UI elements
  - Conveys professionalism and trust

- **Secondary Color: Light Blue (#4A90E2)**
  - Use for secondary buttons, links, and highlights
  - Provides a softer contrast to the primary color

- **Accent Color: Orange (#F5A623)**
  - Use sparingly for calls-to-action, notifications, or important alerts
  - Adds a touch of energy and draws attention to key elements

- **Background Colors:**
  - Light Gray (#F5F5F5) for main background
  - White (#FFFFFF) for content areas and cards

- **Text Colors:**
  - Dark Gray (#333333) for primary text
  - Medium Gray (#666666) for secondary text

### Fonts:
- **Headers: Roboto**
  - A clean, modern sans-serif font
  - Use for main headings, section titles, and navigation items

- **Body Text: Open Sans**
  - Highly readable sans-serif font
  - Use for general content, descriptions, and form labels

- **Monospace Font: Source Code Pro**
  - Use for displaying code snippets or employee IDs

### Where to use these elements:
- **Login Screen:**
  - Navy blue header with white text
  - White login form on a light gray background
  - Orange "Login" button

- **Dashboard:**
  - Navy blue top navigation bar
  - Light gray background with white content cards
  - Roboto for section titles
  - Open Sans for statistics and summaries

- **Employee List:**
  - White background with alternating light gray rows for better readability
  - Navy blue column headers
  - Light blue for clickable employee names

- **Forms (e.g., Add/Edit Employee):**
  - White background
  - Navy blue section dividers
  - Light blue form field highlights on focus
  - Orange "Submit" button

- **Notifications and Alerts:**
  - Use orange for important notifications
  - Light blue for informational messages

- **Reports and Analytics:**
  - White background
  - Use the color scheme (navy, light blue, orange) for charts and graphs
  - Roboto for chart titles and axis labels
  - Open Sans for data points and legends
